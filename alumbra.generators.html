alumbra.generators

operation

(operation schema & [opts])

Create a function that can be called with an operation type (e.g. :query, :mutation, :subscription), as well as the name of the operation, and will produce an operation matching the given analyzed schema.

(def schema
  (alumbra.analyzer/analyze-schema
    "type Person { ... } ... "
    alumbra.parser/parse-schema))

(def gen-operation
  (operation schema))

(gen/sample (gen-operation :query "Q"))

schema must conform to :alumbra/analyzed-schema (see alumbra.spec).
